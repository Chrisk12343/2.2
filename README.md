# Browser-Based Interactive Project — CSC Assignment

## Overview

This project is a collection of small browser-based interactive activities built using HTML, CSS, and JavaScript. It demonstrates basic front-end development concepts such as user interaction, styling, and organizing multiple components within a single project.

## Features

- Multiple interactive mini-activities  
- JavaScript-based user interaction  
- Custom styling with CSS  
- Organized project structure with separate components  

## Project Structure

- `game.html` — main entry point for the project  
- `button.js` — JavaScript logic for interactions  
- `game.css` — styling for the main page  

### Activity Folders

- `number 1` — first mini-activity with its own assets and HTML  
- `number 2` — additional activity or variation  
- `number 3` — supporting content or extended functionality  
- `number 4` — extra resources or test components  

## Requirements

- Any modern web browser  

## How to Run

- Open `game.html` in your browser  

## Notes

- No build tools or frameworks required — pure HTML, CSS, and JavaScript  
- Each numbered folder contains additional content that can be explored individually  
- Project structure is organized to separate different activities and assets  

## What I Learned

- Handling user interaction with JavaScript  
- Structuring multi-part front-end projects  
- Organizing files for readability and scalability  
- Applying CSS for consistent styling across pages
